# AI and Machine Learning Projects

A comprehensive repository containing a collection of **Artificial Intelligence and Machine Learning** projects implemented using Python and Jupyter Notebooks. This repository covers fundamental concepts, applied models, deep learning techniques, and real-world problem-solving use cases.

---

## Overview

This repository is organized into multiple folders, each representing a focused AI or Machine Learning concept or project. The notebooks include experiments, model implementations, and analytical workflows designed for learning, practice, and portfolio development.

---

## Projects Included

- Activation Functions (Neural Networks)  
- Capstone Project – Trainee Full Stack Developer  
- Credit Card Fraud Detection  
- Fine-Tuning a Small Language Model  
- How a Neuron Works  
- Image Augmentation  
- Image Classification using Transfer Learning  
- Keyboard Auto Segmentation (NLP)  
- Laptop Price Predictor  
- Polynomial Regression using Python  
- Regression Neural Network  
- Regression Problems with Neural Networks  
- Vector Search with ChromaDB  
- Vector Search with Pinecone  
- Water Quality Prediction ML Model
